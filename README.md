# ThreadLibrary

This is a Preemptive User level Thread Library which simulates green threads using their core functions like create,join,yield etc. 

I have also implemented mutexes (Mutually exclusive locks),Round Robin Scheduler, data structures, event handlers, manual yield functionalities and several benchmarks(metrics). 

Benchmarks are for testing against the original pthreads library vs this thread library!

***Feel free to look at the often occuring comments I added throughout the program for better understanding of the workflow but keep in mind that they most of the time are written very vaguely and crisp and sometimes slightly misleading to actual factual description of the method you might find on the web.***
